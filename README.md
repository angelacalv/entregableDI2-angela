Viajeros Aventureros 

Proyecto DI — Entrega 2 y 3 (IES Laguna de Tollón · 2025/2026)

El proyecto es de una aplicación de viajes donde se puede ver información de diferentes destinos.

Esta aplicación web ha sido desarrollada aplicando los conocimientos adquiridos en la unidad sobre creación de interfaces funcionales con React/Next.js y Tailwind CSS, a partir del prototipo diseñado previamente en Figma.

La aplicación es completamente responsive, adaptada a escritorio y dispositivos móviles.

El proyecto incluye distintas pantallas: landing, inicio de sesión, registro, explorador de destinos, detalle de cada destino, viajes guardados, área de usuario y panel de administrador (simulado).
